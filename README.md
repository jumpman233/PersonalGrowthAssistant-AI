# Personal Growth Assistant

Nuxt 4 + TypeScript project scaffolded with pnpm, ESLint, Prettier, and Tailwind CSS.

## Common Commands

```bash
# Install dependencies
pnpm install

# Start the local development server
pnpm dev

# Run ESLint checks
pnpm lint

# Format files with Prettier
pnpm format

# Build for production
pnpm build

# Preview the production build
pnpm preview
```
