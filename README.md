# Personal Growth Assistant

Nuxt 4 + TypeScript project scaffolded with pnpm, ESLint, Prettier, and Tailwind CSS.

## Common Commands

```bash
# Install dependencies
pnpm install

# Start the local development server
pnpm dev

# Run ESLint checks
pnpm lint

# Format files with Prettier
pnpm format

# Build for production
pnpm build

# Preview the production build
pnpm preview

# sql 启停命令1
pg_ctl -D E:\software\postgreSQL\postgresql-18.3-3-windows-x64-binaries\pgsql\data -l pg_log.log start

# SQL注册windows服务
pg_ctl register -D D:\pgsql\data -N postgresql-17 -U LocalSystem

# 启动windows的postgresql服务
net start postgresql-17
# 停止windows的postgresql服务
net stop postgresql-17

# 登录postgresql
psql -U postgres
```
