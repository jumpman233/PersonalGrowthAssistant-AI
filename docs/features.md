# 功能清单

## 1. 记录模块

### 1.1 新建记录

字段：

- 标题
- 内容
- 分类
- 心情评分
- 真实建设感评分
- 消耗程度评分
- 标签
- 创建时间

分类：

- 职业
- 关系
- 情绪
- 学习
- 生活
- 项目

### 1.2 记录列表

功能：

- 展示所有记录
- 按时间倒序
- 显示标题、摘要、分类、评分、标签
- 点击进入详情

### 1.3 记录详情

功能：

- 查看完整内容
- 查看评分
- 查看标签
- 查看 AI 总结
- 编辑
- 删除

---

## 2. AI总结模块

### 2.1 单条记录总结

输入：

- 标题
- 内容
- 分类
- 心情评分
- 建设感评分
- 消耗评分

输出：

- 事件摘要
- 当前情绪
- 主要消耗来源
- 真实建设感来源
- 下一步建议

### 2.2 周复盘总结

输入：

- 本周所有记录

输出：

- 本周主要推进
- 本周主要消耗
- 高频情绪
- 高频标签
- 下周一个最小行动

---

## 3. 标签与分类模块

功能：

- 创建标签
- 自动从记录中提取标签
- 手动编辑标签
- 按分类筛选
- 按标签筛选

---

## 4. 数据反馈模块

第一版只做简单版：

- 本周记录数
- 平均心情分
- 平均建设感分
- 平均消耗分
- 高频标签
- 最近7天趋势

---

## 5. 设置模块

功能：

- 用户昵称
- 个人复盘偏好
- AI输出风格
- 常用复盘模板

第一版可以先不做完整设置，只在代码里写默认模板。
