# AI辅助成长复盘系统 PRD

## 1. 产品定位

一个帮助用户记录职业、关系、情绪和项目事件，并通过标签、评分、AI总结和趋势反馈识别长期模式的成长复盘工具。

## 2. 目标用户

初期目标用户就是我自己：

- 正在职业重启
- 需要恢复稳定输出
- 需要记录每天的真实建设感
- 需要识别消耗来源
- 需要观察关系/职业/情绪模式
- 希望用 AI 帮助自己复盘，而不是替自己做决定

## 3. 核心问题

用户不是缺少记录工具，而是缺少一种能把记录转化为反馈的系统。

要解决的问题：

1. 每天做了很多事，但不知道哪些真正有建设感
2. 容易被琐碎事情消耗
3. 情绪、职业、关系问题容易混在一起
4. 复盘时容易空泛
5. 需要一个低压力的方式记录自己如何恢复稳定输出

## 4. 核心价值

本产品帮助用户完成：

记录事件 → 标记感受 → 识别模式 → 生成反馈 → 调整行动

## 5. MVP范围

第一版只做：

1. 新建记录
2. 记录列表
3. 记录详情
4. 分类标签
5. 心情评分
6. 真实建设感评分
7. 消耗程度评分
8. AI总结
9. 简单周复盘

## 6. 暂不做

第一版暂不做：

1. 社交功能
2. 多人协作
3. 复杂图表
4. 复杂权限
5. 移动端深度适配
6. 完整心理测评
7. 诊断/治疗相关功能

## 6. 设计风格

1. Notion 的清爽
2. Day One 的记录感
3. Linear 的秩序感
4. Apple Health 的趋势反馈
5. 少量卡片式布局

## 7. 文案风格

不要：
- 你今天必须完成复盘
- AI帮你诊断心理问题
- 立即改变人生

要：
- 记录一点就够了
- 看看今天什么真正推动了你
- 不急着解决，先看清楚
- 找到一个明天能做的小行动
